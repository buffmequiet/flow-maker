# Flow Maker — Plan

## 1. 서비스 개요

호텔 객실 QR 스캔 → 외국인 관광객 대상 K뷰티·마사지·택시·밴 예약 플랫폼.
본 프로젝트(Flow Maker)는 **운영자(모리스) 어드민** — 제휴업체 등록 및 관리 기능을 구현한다.

### 전체 서비스 플로우

```
관광객 (호텔 객실 QR 스캔)
    ↓ 앱 설치 없이 모바일 웹 접속
서비스 화면 (K뷰티 / 마사지 / 택시 / 밴 등 선택)
    ↓ 예약 요청
운영자(모리스) 어드민 → 제휴업체에 직접 연락 → 예약 대행
    ↓ (장기 목표)
제휴업체가 직접 수락/거절 (자동화)
```

---

## 2. 타겟

- **사용자 A**: 외국인 관광객 — 호텔 객실 QR 스캔 후 예약 (본 프로젝트 범위 외)
- **사용자 B (본 프로젝트)**: 운영자(모리스) — 어드민에서 제휴업체 등록·관리

---

## 3. 비즈니스 맥락

- 운영자가 직접 제휴업체를 섭외하고 수수료 기반으로 예약을 중개
- 현재는 수동으로 업체 연락처를 관리 → 체계화 필요
- 장기적으로는 업체가 어드민 계정을 갖고 직접 수락/거절하는 구조로 자동화 예정

---

## 4. 핵심 기능 (이번 범위)

### 4-1. 제휴업체 등록
- 업체명 (한글 / 영어)
- 서비스 유형 (K뷰티 / 마사지 / 택시 / 밴 / 기타 — 추후 유형 추가 가능)
- 대표 이미지 (1장 이상)
- 연락처 (전화번호, 카카오톡 ID 등)
- 운영시간 (요일별 설정 가능)
- 가격표 (항목명 + 금액, 복수 항목)
- 제휴 수수료 (%)
- 메모 (내부용)

### 4-2. 제휴업체 목록
- 전체 등록 업체 리스트
- 서비스 유형별 필터
- 업체 활성/비활성 토글

### 4-3. 업체 상세 / 수정
- 등록 정보 조회
- 수정 및 저장

### 4-4. 서비스 유형 관리 (기초)
- 유형 목록 보기
- 유형 추가
